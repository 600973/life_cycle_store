## Обзор проекта

Проект на Python/Jupyter Notebook для анализа еженедельных продаж по нескольким магазинам (8 магазинов, 310 недель). Выполняет статистический анализ и создаёт интерактивные визуализации.

## Запуск проекта

```bash
# Установка зависимостей
pip install pandas numpy plotly scipy ipywidgets

# Запуск ноутбука
jupyter notebook Untitled.ipynb
```

Ноутбук ожидает файл `weekly_sales.xlsx` в корне проекта со столбцами: «Магазин» и столбцами еженедельных продаж.

## Архитектура

Проект находится в одном ноутбуке (`Untitled.ipynb`) из 8 ячеек, образующих следующий конвейер:

1. **Импорты** — pandas, numpy, plotly, scipy, ipywidgets
2. **Загрузка данных** — чтение `weekly_sales.xlsx` в DataFrame
3. **Класс `WeeklySalesAnalyzer`** — основной аналитический движок, вычисляющий 5 метрик:
   - Рост в % (изменение неделя к неделе)
   - Индекс роста (относительно первой недели, базовая линия 100%)
   - Накопительная сумма
   - Скользящее среднее за 4 недели
   - Линейный тренд с R² (через `scipy.stats.linregress`)
4. **`create_weekly_chart()`** — построение графиков на Plotly с опциями: линии тренда, коридоры мин-макс, наложение среднего/медианы
5. **Интерактивный дашборд** — элементы управления `ipywidgets` (выбор метрики, мультивыбор магазинов, слайдер диапазона недель, переключатели отображения) с обратными вызовами `observe()` для обновления в реальном времени
6. **Отображение таблицы данных**
7. **Сводная статистика**
8. **`export_to_html()`** — генерация автономного HTML с встроенным Plotly.js и данными в формате JSON (сервер не нужен)

## Примечания

- Комментарии и подписи в коде на русском языке
- Формальных тестов, линтинга и системы сборки нет — это аналитический ноутбук
- HTML-экспорт (`weekly_sales_chart.html`) — сгенерированный вывод, а не исходный код
